# Módulo 01: Fundamentos de Pesquisa Operacional e Otimização

Bem-vindo ao primeiro módulo do curso de Otimização! Este material foi desenhado para introduzir conceitos matemáticos complexos através de aplicações práticas e programação em Python.

![Resumo de Problemas de Otimização](img/summary.png)

## Conteúdo do Módulo

### [Aula 01: Introdução à Programação Linear](aula_01_introducao.ipynb)
- O que é Pesquisa Operacional.
- Formulação de modelos lineares.
- Resolução de problemas de mix de produção (A Fábrica de Móveis).
- Introdução à biblioteca PuLP.

### [Aula 02: Problema do Caixeiro Viajante (TSP)](aula_02_tsp.ipynb)
- Modelagem de problemas combinatórios.
- Restrições de eliminação de sub-ciclos (MTZ).
- Visualização de rotas ótimas.
- Testes de performance com instâncias maiores.

### [Aula 03: Problemas de Transporte e Redes](aula_03_transporte_redes.ipynb)
- Otimização logística entre fábricas e centros de distribuição.
- Visualização de fluxos em rede com espessura dinâmica.
- Análise de sensibilidade (custos e restrições).

### [Aula 04: Localização de Instalações](aula_04_localizacao.ipynb)
- O Problema das P-Medianas e P-Hub.
- Decisões estratégicas de infraestrutura.
- Clustering e atendimento de demanda ponderada.

## Como usar este material?
1. Certifique-se de ter o Python instalado.
2. Instale as dependências: `pip install pulp matplotlib`.
3. Siga a ordem numérica das aulas.
4. Realize os exercícios propostos em cada notebook para fixar os conceitos.
